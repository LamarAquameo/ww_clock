# ww_clock
  倒计时插件
#插件介绍
  1、该插件基于jquery开发;
  
  2、引入jquey.js与ww.min.jquery.js;
  
  3、在html里面写一个放倒计时的div节点;

    <div class="ww1"></div>
  
  4、初始化封装的函数;

     ww_clock("1488899999",$('.ww1'),"ww_1");

#注释：
第一个参数为时间戳，第二个参数为放倒计时的节点，第三个参数为展示样式；
#展示样式有四种：
  ww_1：241天02时49分19秒；
  
  ww_2：241天02:49:19；
  
  ww_3：241天02-49-19；
  
  ww_4：241天02/49/19；
